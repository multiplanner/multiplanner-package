# 🚆 Multiplanner

Multiplanner is een NodeJS-libary voor het plannen van uw treinrit.

## 👾 Instalatie

Gebruik de package manager [NodeJS](https://nodejs.org/) om Multiplanner te installeren.

```bash
npm install multiplanner
```
